Distribute Module

A module to help distributing objects to a nice formation by creating coordinates you can move you objects to or move them directly.

Arc (numberOfObjects, startAngle, endAngle, radius)
Create a table of coordinates that describe an arc shape An arc is a open circle that a has a start and a end angle to describe that open circle

Parameters:

  • numberOfObjects int the number of objects to distribute
  • startAngle int the starting angle of the arc
  • endAngle int the ending angle of the arc
  • radius int the radius of the arc

    • 0 degrees is looking to the east
    • -90 degrees would be north
    • 90 degrees is south
    • -180 or 180 degress is west

Returns:

    table with position coordinates

Usage:

     local coordinatesTable = Distribute:Arc(55, -90, 90, 150)
ArcObjects (options)
Distribute objects on an arc shape An arc is a open circle that a has a start and a end angle to describe that open circle

Parameters:

  • options

    a table with options

    • tableOfObjects: (Table) a table with the objects to be distributed
    • startAngle: (number) starting angle of the arc (default: 0)
    • endAngle: (number) ending angle of the arc (default: 180)
    • radius: (number) the radius of the circle (default: 150)
    • offsetX: (number) moves the whole formation (default: 0)
    • offsetY: (number) moves the whole formation (default: 0)
    • transition: (transition) a easing transiton (default: easing.inOutQuad)

    • time: (number) the time of the transition (default: 1000)

    • 0 degrees is looking to the east

    • -90 degrees would be north
    • 90 degrees is south
    • -180 or 180 degress is west

Returns:

    nothing

Usage:

     local Distribute = require("devilsquid.distribute.distribute")

 Distribute:ArcObjects({
 	tableOfObjects 	= myTableOfObjects,
 	startAngle 		= -90,
 	endAngle 		= 90,
 	radius 			= 150,
 	offsetX 		= display.contentCenterX,
 	offsetY 		= display.contentCenterY,
 	transition 		= easing.inOutBounce,
 	time 			= 1200
 	})
